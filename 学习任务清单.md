## 📋 任务清单

### 阶段一：基础理解（2-3小时）

#### 任务 1.1：理解核心概念 ⭐⭐⭐
- [ ] 阅读 `reasoners/base.py`（30分钟）
  - 理解 WorldModel、SearchConfig、SearchAlgorithm 的作用
  - 理解状态、动作、奖励的概念
- [ ] 阅读 `代码库学习指南.md`（20分钟）
  - 理解整体架构
  - 理解设计理念

**检查点**：你能回答以下问题吗？
- WorldModel 是干什么的？
- SearchConfig 是干什么的？
- fast_reward 和 reward 有什么区别？

#### 任务 1.2：看一个简单示例 ⭐⭐⭐
- [ ] 阅读 `examples/CoT/blocksworld/world_model.py`（20分钟）
  - 看如何定义状态转换
- [ ] 阅读 `examples/CoT/blocksworld/search_config.py`（20分钟）
  - 看如何定义奖励函数
- [ ] 阅读 `examples/CoT/blocksworld/inference.py`（15分钟）
  - 看如何组合使用

**检查点**：你能解释积木问题的状态转换吗？

#### 任务 1.3：理解搜索算法 ⭐⭐
- [ ] 阅读 `reasoners/algorithm/mcts.py`（45分钟）
  - 理解 MCTS 的四个阶段
  - 理解 UCT 公式

**检查点**：你能解释 MCTS 如何工作吗？

## 🚀 快速开始（5分钟）

### 最快上手方式

```bash
# 1. 安装
conda create -n reasoners python=3.10
conda activate reasoners
pip install -e .

# 2. 运行示例（使用 API，无需 GPU）
python examples/CoT/gsm8k/inference.py \
    --base_lm openai \
    --model_dir "gpt-4" \
    --temperature 0
```
